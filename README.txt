
This project is a direct port, from Java to Scala 2.8, of
the sample code of the book "Growing Object Oriented Software,
Guided By Tests" by Steve Freeman and Nat Pryce.

The port to Scala 2.8 was done by Spiros Tzavellas.

The book's website:
  http://www.growing-object-oriented-software.com/

The original code:
  http://github.com/sf105/goos-code/
